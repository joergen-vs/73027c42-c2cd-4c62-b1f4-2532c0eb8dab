# charset

```xml
<xs:element name="charset" type="xs:string"/>
```

Akseptert tegnsett for fil.

Eksempel:
```xml
<charset>right</charset> 
```

## Verdier
Nav på tegnsett. Lovlige verdier i Norge inkluderer
* ISO 8859-1
* ISO 8859-4
* UTF-8
